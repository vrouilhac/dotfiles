# Dotfiles

## Installation

Do not install xD

## Full setup requirements

Install:
	- tmux
	- vim
		- pathogen
		- vim-plug
	- zsh
	- rustup
	- go
	- glow
	- ripgrep
	- fzf

Rust setup
	- add 'rls' component
	- add 'rust-analysis' component

Go setup
	- install gopls go install golang.org/x/tools/gopls@latest

Nodejs setup
	- install nodejs
	- install typescript globally
